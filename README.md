# Inventory Management System

**Author**: Natalia Pozo
**Bootcamp**: Adalab - Data Analytics (Module 1: Python)

## Description
This project is part of the final evaluation for Module 1. It consists of a Python-based system to manage a store inventory, track clients, and process sales using fundamental programming concepts (functions, loops, and data structures).

## Structure
- `inventory_management.ipynb`: Main notebook containing the source code and logic.

## How to run
1. Clone the repository.
2. Open `inventory_management.ipynb` in Jupyter Notebook or VS Code.
3. Run the cells to interact with the inventory system.
